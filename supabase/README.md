# Supabase Local Development

The local stack includes migrations through Phase 13: identity, governed course content, learning records, AI grading, writing, listening/speaking, administration, API boundaries, notifications, offline sync, and the vocabulary/grammar knowledge API.

Run locally after installing the Supabase CLI:

```bash
supabase start
supabase db reset
```

The remaining text-conversation session schema is documented in `docs/database-schema.md` and will be implemented in a later phase.
